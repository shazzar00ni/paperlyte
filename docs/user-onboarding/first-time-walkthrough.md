2. **Type a title** in the title field

## Step 6: Search Your Notes (1 minute)

Now let's practice finding notes quickly.

### Action Items:

1. **Click the search box** (or press `Ctrl+F` / `Cmd+F`)

2. **Type: "shopping"**
   - Your "Shopping List" note appears instantly

3. **Clear search and try: "ideas"**
   - "Work Ideas" note shows up

4. **Search by tag: "tag:personal"**
   - All notes with the `personal` tag appear

### Search Tips:
- Search works across titles, content, and tags
- Results appear as you type
- Search is case-insensitive
