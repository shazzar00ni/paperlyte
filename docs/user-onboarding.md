## ðŸš€ Quick Start (30 Seconds)

1. **Create your first note** - Click "New Note" or press `Ctrl+N` / `Cmd+N`
2. **Start typing** - Your note saves automatically
3. **Add tags** - Organize with tags like `#work` or `#personal`
4. **Search anything** - Press `Ctrl+F` / `Cmd+F` to find notes instantly

That's it! You're ready to start using Paperlyte.

## ðŸ’¡ Core Concepts

### Notes

The fundamental unit in Paperlyte. Each note has:
- **Title** - A descriptive name for quick identification
- **Content** - Rich text or markdown formatted content
- **Tags** - Organizational labels for grouping and filtering
- **Metadata** - Created/modified dates, word count, etc.

### Tags

Flexible organization system that allows:
- Multiple tags per note
- Hierarchical organization (`project:feature:detail`)
- Quick filtering and searching
- Status tracking (`#todo`, `#done`)

### Search

Powerful instant search across:
- Note titles
- Note content
- Tags
- Metadata

### Auto-Save

Everything saves automatically:
- Changes save as you type
- No "save" button needed (but `Ctrl+S` / `Cmd+S` works too)
- Never lose your work

### Common First-Time Issues

**"Where's the save button?"**
- There isn't one! Everything saves automatically as you type.

**"How do I organize my notes?"**
- Use tags! Add tags by clicking the tag icon or typing `#tagname` in your note.

**"Can I use this offline?"**
- Yes! Paperlyte works completely offline. Your notes are stored locally.

**"How do I backup my notes?"**
